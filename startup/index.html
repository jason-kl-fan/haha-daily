Gumloop 完成 5,000 萬美元 B 輪融資，推動企業無程式碼 AI 自動化

AI Agent 落地潮：企業不再滿足於聊天，而是要能串接多種工具的自動工作流。

無程式碼 AI 自動化平台 Gumloop 宣布由 Benchmark 領投的 5,000 萬美元 B 輪融資。 該平台允許非技術人員構建能跨工具執行複雜任務的 AI Agent。 隨著 Perplexity 等巨頭進入 Agent 市場，Gumloop 的彈性與整合能力成為其競爭核心。

Anthropic 取消 Claude 100 萬 token 長文本附加費，並啟動 1 億美元合作夥伴計畫

AI 價格戰轉向長文本：開發成本大幅降低，生態系佈局進入整合期。

Anthropic 宣布其 Claude 4.6 模型取消長文本處理的額外收費，現在處理 100 萬 token 的 context 僅需支付標準 token 費。 同時推出「Clawed Partner Network」並投入 1 億美元，旨在吸引更多系統整合商將 Claude 嵌入企業工作流。

Google 與 Accel 披露 AI 新創趨勢：70% 申請案因『AI 套殼』被淘汰

VC 轉向深科技：僅靠調用 API 的新創已失去吸引力，具備底層研發能力的團隊才是贏家。

Google 與 Accel 的 Atoms 加速器計畫公布最新數據，顯示在 4,000 多件 AI 新創申請案中，絕大多數因缺乏核心技術創新（僅為 LLM 套殼）而遭拒。 入選的 5 家新創皆專注於底層架構與專利模型，反映出創投界對 AI 新創可持續性的嚴格審核。
