微軟 3 月 Patch Tuesday 修補 84 個漏洞，含 2 個已公開零時差

本月企業修補優先序仍是 Windows、SQL Server 與權限提升鏈。

The Hacker News 指出，微軟本月修補 84 個漏洞，其中 46 個是權限提升、18 個是 RCE，另含 .NET 與 SQL Server 兩個已公開漏洞；代表企業仍要先補核心平台。

n8n 修補 2 個 Critical 漏洞，最糟可直接遠端執行命令

研究人員揭露 n8n 兩個已修補漏洞：一個可讓已驗證使用者逃逸 expression sandbox 執行系統命令，另一個則可透過公開表單節點觸發未授權表達式評估。

nx npm 供應鏈事件後續：攻擊者 72 小時內拿下 AWS 管理權

開發者 token 外洩，現在足以一路打穿 GitHub 到雲端身分信任鏈。

Google 威脅報告顯示，UNC6426 利用先前 nx npm 供應鏈攻擊偷來的金鑰，先奪走 GitHub token，再濫用 GitHub 對 AWS 的 OIDC 信任建立新管理角色，最後外洩與破壞生產資料。

CISA 新增 Omnissa Workspace ONE、SolarWinds Web Help Desk 與 Ivanti Endpoint Manager 等 3 個已遭利用漏洞進 KEV，提醒企業應把『有實際利用證據』的漏洞排進最高優先級。
