Apple 緊急發布 WebKit 零日漏洞修補程式 (CVE-2026-20643)

此漏洞涉及跨來源攻擊，可能導致攻擊者在各蘋果作業系統執行惡意代碼。

蘋果今日針對 iOS、iPadOS 及 macOS 推送背景安全更新，修復 WebKit 引擎中一個已被積極利用的關鍵漏洞。官方強烈建議使用者確保設備已開啟自動更新，以防範網路攻擊。

Ubuntu 24.04 爆發高風險 Root 提權漏洞 (CVE-2026-3888)

本地普通使用者可利用此漏洞在預設安裝的桌面版中直接獲取完整系統權限。

安全專家指出該漏洞存在於 Ubuntu 24.04 的核心組件中，風險評分極高。Canonical 已釋出修補更新，伺服器版雖受影響較小但仍建議所有管理員立即排程更新。

Cisco 安全防火牆發現 CVSS 10 分滿分漏洞，遭勒索軟體利用

Interlock 勒索軟體正積極鎖定 Cisco FMC 軟體中無需驗證即可執行 root 指令的瑕疵。

此漏洞 (CVE-2026-20131) 允許遠端攻擊者繞過驗證並執行 Java 代碼，進而完全控制網路設備。Cisco 已發布修補版本，呼籲企業務必優先處理此核心網路防禦缺口。
