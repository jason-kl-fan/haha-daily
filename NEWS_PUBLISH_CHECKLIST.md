# NEWS_PUBLISH_CHECKLIST.md

每次新聞搜尋、整理、推送後，只有全部打勾才算完成。

## A. 本地內容
- [ ] 今天日期頁已建立 / 更新
- [ ] 首頁已改成今天入口，不是昨天
- [ ] 相關 topic 頁已更新
- [ ] `/saved/` 功能需要的資源沒有缺

## B. 推送 / 發布
- [ ] 變更已推到遠端
- [ ] GitHub Pages / 部署流程成功完成
- [ ] 沒有卡在舊 commit、失敗 build、或未部署狀態

## C. 公開站驗證（必要，不能跳過）
- [ ] 公開首頁 `/` 已是今天版本
- [ ] 公開當天頁 `/YYYY-MM-DD/` 可正常開啟
- [ ] 至少檢查一個這次有改到的 topic 頁
- [ ] 公開 `/saved/` 正常

## D. 回報前最後確認
- [ ] 我檢查的是「公開網址」不是本地檔案
- [ ] 我檢查的是「最新上線版本」不是舊快取 / 舊頁面
- [ ] 以上全部完成後，才可以說「好了」

## 硬規則
如果 B 或 C 有任何一項沒完成，就不能宣稱完成，只能如實回報卡在哪裡。
