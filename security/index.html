SAP 修補 FS-QUO 與 NetWeaver 關鍵漏洞

企業核心系統（ERP、保險、門戶）的漏洞修補仍然是企業防護的最前線。

SAP 在 3 月份的安全修補日解決了包含保險報價管理（FS-QUO）與 NetWeaver 企業門戶在內的多項關鍵弱點，強調了大型企業維持系統更新的重要性。

OpenAI Codex Security 正式上線，一個月內找出上萬個高風險漏洞

AI 原生安全代理人正從『掃描工具』進化為可自動化發現與修復漏洞的 AI 工員。

OpenAI 推出 Codex Security 掃描器，首月測試即在 120 萬次 commit 中找出超過 1 萬個高影響力漏洞，顯示 AI 在 DevSecOps 中的強大潛力。

Google 報告指出 2025 年半數零日漏洞鎖定企業技術

駭客攻擊重心正從個人裝置全面轉向擁有高價值資料的企業基礎設施。

Google 最新研究顯示，去年追蹤到的零日漏洞有一半集中在企業級軟硬體，反映出攻擊者正採取更具針對性的商業間諜或勒索策略。
