微軟 3 月 Patch Tuesday 修補 82 個漏洞

企業 Windows 與 Microsoft 服務面仍是本月修補重點，EoP 與 RCE 風險特別高。

CrowdStrike 分析指出，微軟 3 月安全更新涵蓋 82 個漏洞，其中含 8 個 Critical 與 2 個已公開漏洞；Windows 與 Azure 仍是主要修補面向。

來源：CrowdStrike · 原文連結

SAP 修補 FS-QUO 與 NetWeaver 相關高風險漏洞

SAP 核心系統漏洞仍然是大型企業最不能拖延的修補項。

Gemini 搜尋彙整顯示，SAP 3 月 Patch Day 處理了 NetWeaver 與相關產品的關鍵與高優先級漏洞，包含反序列化、SQL Injection 與授權檢查缺失等問題。

來源：SecurityWeek / Gemini web_search · 原文連結

全球網攻強度仍維持高檔，2 月平均每組織每週 2,086 次攻擊

整體攻擊面壓力沒有明顯降溫，企業應把高強度攻擊視為常態。

Check Point 指出，2026 年 2 月全球平均每個組織每週遭遇 2,086 次攻擊，年增 9.6%，即使勒索活動放緩，總體威脅仍維持接近歷史高點。

來源：Check Point Research · 原文連結
